A fun Roman-themed matching app built with React, TypeScript, and Tailwind CSS.

## Setup & Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - The app will be running at `http://localhost:5173`
   - Vite will display the exact URL in the terminal

## Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build locally

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Features

- Random Roman name and activity matching
- Fun loading animations
- Roman-themed UI with pillars and classical design
- Responsive design
